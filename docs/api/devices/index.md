| Категория | Классы | Страница |
|-----------|--------|----------|
| Базовые | `BaseEntity`, `OnOffEntity`, `SimpleSensor` | [Базовые классы](base.md) |
| Освещение | `LightEntity`, `LedStripEntity` | [Освещение](light.md) |
| Климат | `ClimateEntity`, `HvacRadiatorEntity`, `HvacBoilerEntity`, `HvacFanEntity`, `HvacHeaterEntity`, `HvacUnderfloorHeatingEntity`, `HvacAirPurifierEntity` | [Климат](climate.md) |
| Шторы/ворота | `CurtainEntity`, `WindowBlindEntity`, `GateEntity` | [Шторы и ворота](covers.md) |
| Реле/розетки | `RelayEntity`, `SocketEntity`, `ValveEntity`, `ScenarioButtonEntity` | [Реле и розетки](switches.md) |
| Сенсоры | `SensorTempEntity`, `HumiditySensorEntity`, `MotionSensorEntity`, `DoorSensorEntity`, `WaterLeakSensorEntity`, `GasSensorEntity`, `SmokeSensorEntity` | [Сенсоры](sensors.md) |
| Медиа | `TVEntity`, `IntercomEntity` | [Медиа](media.md) |
| Бытовая техника | `KettleEntity`, `VacuumCleanerEntity`, `HumidifierEntity` | [Бытовая техника](appliances.md) |
