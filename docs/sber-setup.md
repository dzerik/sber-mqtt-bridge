# Настройка Sber Studio

Перед установкой интеграции вам нужно получить MQTT-учётные данные от Sber. Это делается через портал Sber Studio.

## Шаг 1: Регистрация в Sber Studio

1. Перейдите на [Sber Studio](https://developers.sber.ru/studio/workspaces/)
2. Войдите с вашим **Sber ID** -- это тот же аккаунт, что используется в приложении Сбер Умный дом
3. Создайте рабочее пространство (workspace), если его ещё нет

!!! info "Sber ID"
    Sber ID -- это единый аккаунт для всех сервисов Сбера. Если вы уже пользуетесь приложением Сбер Умный дом, у вас уже есть Sber ID.

## Шаг 2: Создание проекта интеграции

1. В Sber Studio перейдите в раздел **Умный дом**
2. Нажмите **Создать проект**
3. Выберите тип интеграции: **MQTT-интеграция**
4. Дайте проекту имя, например: `Home Assistant Bridge`

!!! tip "Совет"
    Имя проекта используется только для вашего удобства в Sber Studio. Оно не отображается в приложении Сбер Умный дом.

## Шаг 3: Получение MQTT-учётных данных

1. Откройте настройки созданного проекта
2. Найдите раздел **MQTT-подключение**
3. Скопируйте:
    - **Логин** (MQTT Login)
    - **Пароль** (MQTT Password)

Параметры подключения (заполнены автоматически):

| Параметр | Значение |
|----------|----------|
| Адрес брокера | `mqtt-partners.iot.sberdevices.ru` |
| Порт | `8883` (TLS) |
| Протокол | MQTT over TLS |

!!! warning "Безопасность"
    Храните MQTT-логин и пароль в безопасном месте. Они дают полный доступ к управлению вашими устройствами через облако Sber.

## Шаг 4: Привязка в приложении Сбер

После настройки интеграции в Home Assistant необходимо привязать её в мобильном приложении:

1. Откройте приложение **Сбер Умный дом** на телефоне
2. Перейдите в **Настройки** > **Подключенные сервисы**
3. Ваша MQTT-интеграция должна появиться в списке -- включите её
4. Устройства появятся в приложении после подключения моста (обычно 10-30 секунд)

!!! note "Порядок действий"
    Привязку в приложении Сбер можно делать как до, так и после настройки интеграции в Home Assistant. Устройства появятся только после успешного подключения моста.
