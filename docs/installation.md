# Установка

## Требования

- Home Assistant 2024.1 или новее
- HACS (рекомендуется) или ручная установка
- Учётные данные MQTT из [Sber Studio](sber-setup.md)

## Установка через HACS (рекомендуется)

1. Откройте **HACS** в Home Assistant
2. Нажмите меню (три точки) :material-dots-vertical: > **Пользовательские репозитории**
3. Добавьте репозиторий:
    - URL: `https://github.com/dzerik/sber-mqtt-bridge`
    - Категория: **Интеграция**
4. Найдите **"Sber Smart Home MQTT Bridge"** и нажмите **Установить**
5. **Перезагрузите Home Assistant**

!!! tip "Совет"
    После установки через HACS интеграция будет автоматически обновляться при выходе новых версий.

## Ручная установка

1. Скачайте [последний релиз](https://github.com/dzerik/sber-mqtt-bridge/releases)
2. Распакуйте архив
3. Скопируйте папку `custom_components/sber_mqtt_bridge/` в каталог `config/custom_components/` вашего Home Assistant
4. Перезагрузите Home Assistant

!!! warning "Важно"
    При ручной установке вам нужно будет самостоятельно отслеживать и устанавливать обновления.

## Проверка установки

После перезагрузки Home Assistant:

1. Перейдите в **Настройки** > **Устройства и службы** > **Добавить интеграцию**
2. В поиске введите **"Sber"**
3. Если интеграция **"Sber Smart Home MQTT Bridge"** появилась в списке -- установка прошла успешно

## Следующий шаг

Перейдите к [настройке Sber Studio](sber-setup.md) для получения MQTT-учётных данных, если ещё не сделали этого. Затем -- [настройка интеграции](configuration.md).
